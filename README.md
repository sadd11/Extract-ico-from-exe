# 🖼️ EXE Icon Extractor

A lightweight, modern Desktop application written in Python and CustomTkinter that allows you to easily extract high-quality icons from Windows executable files (`.exe`) and save them as **PNG** or **ICO**.

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Python](https://img.shields.io/badge/python-3.8+-blue.svg)
![Platform](https://img.shields.io/badge/platform-Windows-lightgrey.svg)

---

## ✨ Features

* **Modern UI:** Clean interface built with CustomTkinter following modern UI design principles.
* **Instant Preview:** View extracted icons directly inside the application before saving.
* **Multiple Formats:** Save extracted icons as lossless **PNG** images or multi-resolution **ICO** files.
* **Custom Resolutions:** Support for extracting icons in standard sizes (`16x16`, `32x32`, `48x48`, `64x64`, `128x128`, `256x256`).
* **High-Quality ICO Output:** Automatically packs all standard size variants into saved `.ico` files for seamless Windows scaling.
